# Мой репозиторий devops-netology

## .gitignore правила

Благодаря добавленному файлу terraform/.gitignore будут игнорироваться следующие файлы и каталоги:

Все файлы с расширением .tfstate (например, myinfra.tfstate);

Все файлы, имя которых заканчивается на .tfstate. с любым дополнительным расширением (например, prod.tfstate.backup);

Все файлы с расширением .tfvars (например, secrets.tfvars);

Каталог .terraform (включая всё его содержимое), независимо от уровня вложенности;

Все файлы с расширением .tfplan;

Все файлы с расширением .log;

Все файлы, имя которых начинается с crash.log;

Все файлы с расширением .zip;

Все файлы с расширением .override.tf, .override.tf.json, а также любые файлы, содержащие .override перед расширением .tf или .tf.json.
